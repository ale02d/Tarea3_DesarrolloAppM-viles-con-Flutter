# Semana 3 - Construcción de Wireframe

Desarrolla una interfaz en Flutter utilizando widgets estructurados como AppBar y BottomNavigationBar, además de crear al menos dos widgets personalizados, aplicando buenas prácticas de organización y reutilización de código.

# Estructura
La aplicación fue realizada utilizando Scaffold como estructura principal, lo que permitió organizar el AppBar, el contenido principal y el BottomNavigationBar. Dentro del cuerpo se usó un Column para distribuir los elementos y un Row para el buscador superior, apoyándose en Padding y SizedBox para manejar el espaciado. Además, se crearon widgets personalizados separados en archivos distintos para mantener el código organizado todo siguiendo la estructura aprendida en la clase y la documentación de Flutter.
